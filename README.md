umeditor-turquoise
=======

Fork 自 百度编辑器Ueditor 的迷你版 campaign/umeditor , 做了些样式修改和一点js控制的修改：

1. 修改下拉菜单由单击改为hover显示
2. 修改主题样式， 主色调修改为 turquoise ，按钮由图片全部改为 webfont-icon,.
3. 通过修改stylus文件，可以方便调节主色调。不再局限为 turquoise
4. 兼容性尚未测试， 进行中
5. 请随意fork 和 request pull
